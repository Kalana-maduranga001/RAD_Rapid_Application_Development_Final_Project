# RAD Clothing Store

## Project Description
RAD Clothing Store is a full-stack web application developed as a final project for Rapid Application Development (RAD). The application allows users to browse and purchase clothing products, while admins can manage products and users.

---

## Features
- User authentication (Login/Register)
- User dashboard for managing orders
- Admin dashboard to manage products and users
- Product catalog with categories
- Checkout system
- Responsive UI

---

## Tech Stack
- Frontend: React, TypeScript, Tailwind CSS, React Router
- Backend: Node.js, Express, TypeScript
